1. 리스트 덧셈

    ex) a = [1,2,3], b = [1,1,1]
        -> a + b = [1,2,3,1,1,1]

----------------------------------------------------------------------
2. append(value)
    : 리스트의 마지막에 "값" 넣기

    ex) a.append(3)

----------------------------------------------------------------------
3. insert(index, value)
    : 리스트의 index a에서 b를 넣어라

    ex) a.insert(3, 6)
----------------------------------------------------------------------
4. pop(index)
    : 인자 없을 때 -> 리스트의 마지막 값 꺼내고 리스트에서 삭제
      인자로 index 넘어올 때 -> 해당 인덱스의 값 꺼내고 리스트에서 삭제

    ex) a.pop()     a.pop(3)

----------------------------------------------------------------------
5. remove(value)
    : 리스트에서 해당 "값"을 제거해라

    ex) a.remove(4)

----------------------------------------------------------------------
6. index(value)
    : 리스트에서 value와 같은 값을 찾아 해당 index를 리턴

    ex) a.index(5)
    
----------------------------------------------------------------------
7. r.shuffle(list)
    : 리스트를 섞어라 -> 실제 list가 변경됨
      "import random as r" 을 통해 import 해야함
    
----------------------------------------------------------------------
8. sort(reverse=false)
    : 해당 리스트 오름차순
      reverse=false 가 default임
      true로 설정하면 내림차순

    ex) a.sort() 
        
----------------------------------------------------------------------
9. clear()
    : 리스트 비워버리고, 빈 리스트만 남음

    ex) a.clear() 
        
----------------------------------------------------------------------
10. 슬라이스

    ex) a[:3] : 0번 index에서 2번 index까지
        a[1:4] : 1번 index에서 3번 index까지
               
----------------------------------------------------------------------
11. enumerate(list)
    : 리스트의 index와 값을 한 세트로 묶어서 tuple로 반환
      반복문에서 사용

    ex) for x in enumerate(list):
            print(x[0], x[1])
            --> x[0]은 index, x[1]은 index에 해당하는 값
    
    ex) for idx, value in enumerate(list):
        : 이게 좀 더 직관적
                      
----------------------------------------------------------------------
12. all(...) 
    : 인자로 들어온 것이 "모두" true일 때, all()은 true를 return

    ex) if all(60 > x for x in list):
        : list의 모든 원소가 60보다 작아야 -> all은 true를 리턴
          "모두" 조건을 만족해야 "ture"를 리턴
          
----------------------------------------------------------------------
13. any(...)
    : 인자로 들어온 것 중 "하나라도" true일 때, any()는 true를 return 
      "전부" false인 경우에만 any()는 false를 return

    ex) if any(60 > x for x in list):
        : list 원소 중 하나라도 60보다 작으면 any()는 true를 리턴

----------------------------------------------------------------------
14. 2차원 리스트 생성
    ex) list_a = [ [0]*3 for _ in range(3)]
        : [0]*3 을 3번 반복 실행
        --> list_a = [ [0, 0, 0], [0, 0, 0], [0, 0, 0] ]

    tip) "표"를 통해 이해하는게 좋음
            0 1 2   
          0 0 0 0 -> row 0
          1 0 0 0 -> row 1
          2 0 0 0 -> row 2
          : a[행][열]

----------------------------------------------------------------------
15. 2차원 리스트에서 값에 접근
    ex) list_a = [ [0]*3 for _ in range(3)]
        for x in list_a:
            for y in x:
                print(y)
        
        또는

        for i in range(len(list_a)):
            for j in range(len(list_a[i])):
                print(list_a[i][j])

----------------------------------------------------------------------
