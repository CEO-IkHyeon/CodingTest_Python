1. find()
    : tmp 문자열에서 첫번째로 찾은 'T'의 index

    ex) tmp.find('T')       
------------------------------------------------------------------------
2. count()
    : tmp 문자열에서 'T'가 몇 개 있는지

    ex) tmp.count('T')      
------------------------------------------------------------------------
3. slice 기능
    ex) tmp[:2]     처음부터 1번 index까지
        tmp[3:5]    index 3부터 4까지
        tmp[x:y]    x ~ y-1까지 출력

------------------------------------------------------------------------
4. ord(str)
    : str의 아스키넘버를 return
    
    ex) ord('A')    A의 아스키넘버 65 리턴
        ord('a')    a의 아스키넘버 97 리턴

------------------------------------------------------------------------
5. chr(int)
    : int를 아스키 넘버라고 생각하고 해당하는 알파벳을 return

    ex) chr(65)     아스키넘버 65에 해당하는 알파벳 'A' 리턴

------------------------------------------------------------------------
